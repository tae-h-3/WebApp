### 용어
* Release: 같은 제품을 새롭게 만드는 것(예: 새로운 버전을 배포, 새로운 아이피 번로 부여)
* Deploy: 프로그램 등을 서버와 같은 기기에 설치하여 서비스 등을 제공하는 의미
* Distribute: 제품을 사용자들이 사용할 수 있도록 서비스 등을 제공하는 의미

* nginx 발표 준비
* 리액트 프로젝트 빌드해서 깃허브에 호스팅하기 (레포트)
